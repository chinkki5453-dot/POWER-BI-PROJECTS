Netflix Content Analysis Dashboard — Power BI

Project Overview
This Power BI project delivers an end-to-end analytical dashboard of the Netflix content library. It transforms raw catalog data into interactive visual insights covering content growth, format distribution, geographic spread, genre popularity, and audience ratings. The dashboard is designed to highlight Netflix’s content strategy, regional expansion, and audience targeting patterns through multi-page interactive reports.

Dashboard Structure
The report is organized into multiple interactive pages, each focusing on a specific analytical theme:

1. Executive Summary
	1. High-level KPI overview of the Netflix catalog
	2. Release Year slicer (1942–2021)
	3. Key metrics: 
		Total Titles: 12K 
		Total Movies: 11K 
		Total TV Shows: 367

2. Content Growth Analysis
	1. Line chart showing content additions over time
	2. Highlights sharp growth in releases after 2015–2016
	3. Helps identify platform expansion phases

3. Movies vs TV Shows Distribution
	1. Pie / donut chart comparing content formats
	2. Movies dominate the catalog (~97%)
	3. TV shows form a smaller but high-engagement segment

4. Global Content Distribution
	1. Interactive world map visualization
	2. Country-wise title distribution
	3. Key contributors:	
			1. United States — largest content hub
			2. India — major growth contributor
			3. Supports geographic trend analysis

5. Genre & Audience Analysis
	1. Top genre bar charts and rankings
	2. Most frequent categories include:
	3. International Movies
	4. Dramas
	5. Comedies
	6. Target audience treemap shows rating distribution:
	7. TV-MA and TV-14 form the largest share

Key Insights
	1. Format Dominance: Netflix catalog is heavily movie-driven
	2. Rapid Expansion: Strong rise in content additions in recent years
	3. Global Reach: Content is globally distributed but concentrated in top producer countries
	4. Audience Targeting: Mature audience ratings (TV-MA, TV-14) dominate
	5. Genre Focus: International and drama-based content leads in volume

Tools & Technologies
	1. Power BI Desktop
	2. Power Query — data cleaning & transformation
	3. DAX — calculated measures & KPIs
	4. Bing Maps Visual — geographic distribution
	5. Custom themed visuals (Netflix-style dark theme)

Visualization Types Used
	1. KPI Cards
	2. Line Charts (trend analysis)
	3. Pie / Donut Charts (distribution)
	4. Bar & Stacked Bar Charts (genre & country rankings)
	5. Treemap (audience ratings)
	6. Map Visuals (global distribution)
	7. Slicers (year & type filters)

Interactivity Features
	1. Global Release Year slicer across report pages
	2. Content Type filters (Movie / TV Show)
	3. Cross-filtering between visuals
	4. Clickable map and genre visuals for drill-based exploration

How to Use the Dashboard
	1. Use the Release Year slicer to filter content by time period
	2. Select Movie or TV Show to refine comparisons
	3. Click countries on the map to view regional contributions
	4. Select genres or ratings to dynamically update visuals

Dataset
Netflix titles dataset (kaggle's dataset used for analytics & visualization practice)
